# Homework Assistant

An educational platform designed to help university students quickly find answers to assignments from approved course materials, without having to search manually through multiple files.

## Project Idea

The student selects:

College → Major → Level → Course → Assignment

The student can then enter a question, and the website searches for the answer within the approved content and files that have already been added to the platform.

> If the answer is not available in the approved materials, the website may not return an answer.

## Features

* Simple and user-friendly Arabic interface.
* Courses organized by major and academic level.
* Searches for assignment answers from approved materials.
* Light and dark mode support.
* Admin dashboard for managing courses, assignments, and content.
* Students can use the platform without creating an account.

## Artificial Intelligence

The AI feature is not currently enabled.

It may be added in the future to help answer questions that are not covered in the approved course materials, depending on the usage and demand for the website.

## Project Status

This project is still under development and testing. More improvements and features may be added in the future.

## Technologies Used

* Next.js
* TypeScript
* Tailwind CSS
* Supabase

## Disclaimer

This website is intended to assist students. Users should always review and verify answers before submitting their assignments.
